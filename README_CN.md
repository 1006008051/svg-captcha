# svg验证码

在node.js中生成svg格式的验证码

## 如果你遇到这些问题

- 无法使用 google recaptcha
- 无法安装 c++ 模块

## 为什么使用 svg 格式?

我和我的同事都是用windows系统，不想使用 c++ 模块。  
但现有的验证码生成都是基于 c++ 的。  
svg 格式的验证码竟然比 jpg 格式的还要小。
